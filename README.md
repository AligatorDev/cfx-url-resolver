# cfx-url-resolver





Simple CFX URL resolver

# INSTALL

- Open your console

- Write :
```
  mkdir cfx-resolver
```

- Write :
```
  cd cfx-resolver
```
- Paste the cfx.js in this folder.


Now, return to console


- Write :

```
  npm install request
```

Your cfx-resolver was install! Now let's use it!


# Method to use 

node cfx.js (CFX URL)
